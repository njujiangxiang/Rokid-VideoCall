## 🚀 快速启动

### 1. 启动后端服务器

```bash
cd ~/Desktop/Rokid-VideoCall-Demo
./scripts/start.sh
```

或手动启动：

```bash
cd ~/Desktop/Rokid-VideoCall-Demo/backend-server
npm install
npm run dev
```

### 2. 创建测试用户

```bash
curl -X POST http://localhost:3000/api/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"expert1","password":"123456","role":"expert"}'
```

### 3. 访问管理台

浏览器打开：http://localhost:3000

使用刚创建的账号登录。

### 4. 编译 Android 应用

1. 用 Android Studio 打开 `android-app/` 目录
2. 修改 `Config.kt` 中的服务器地址
3. 添加 Rokid CXR-M SDK 依赖
4. 编译运行
